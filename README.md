# Пакет для ролевой системы проекттов на Symfony

## Руководство разработчика

### 1. Системные требования

- PHP >= 8.2
- Symfony >= 7.1

### 2. Установка

- в основном проекте в composer.json прописать
  ```
  "repositories": [
        {
            "type": "vcs",
            "url": "https://gitlab.dev-u.ru/bundles/logger_crud_bundle.git"
        }
  ]
  ```
- выполнить команду `composer require academcity/logger-crud-bundle:dev-main`
